# Kỹ năng: Đồng bộ Trạng thái Ticket

Kỹ năng này giúp duy trì Dashboard dự án luôn cập nhật mà không cần thao tác thủ công.

## 🎯 When to Use
- Sau khi hoàn thành một task và cập nhật file ticket tương ứng.
- Khi muốn kiểm tra tổng quan tiến độ của Sprint hiện tại.

## 🛠️ How to Use
1. Duyệt qua thư mục `docs/project_managements/ticket/Sprint-XX/`.
2. Đọc trạng thái (Status) và các Task hoàn thành trong từng file `.md`.
3. Cập nhật thông tin tương ứng vào bảng trong `docs/project_managements/ticket/Tickets_Status.md`.
