# Title: /plan-feature
# Description: Nghiên cứu và lập kế hoạch cho một tính năng mới

Quy trình chuẩn để bắt đầu một tính năng mới:

1. **Research:** Tìm kiếm tài liệu, thư viện và các pattern phù hợp.
2. **Analysis:** Sử dụng `logic-lens` để phân tích các yêu cầu và rủi ro.
3. **Architect:** Phác thảo cấu trúc module, schema DB (nếu có).
4. **Implementation Plan:** Tạo file `implementation_plan.md` trong thư mục `docs/project_managements/plans/`.
5. **Review:** Gửi bản kế hoạch cho người dùng và chờ xác nhận trước khi code.
