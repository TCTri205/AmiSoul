# Quản lý Dự án AmiSoul

Thư mục này chứa các tài liệu liên quan đến việc quản lý tiến độ, lập kế hoạch và theo dõi tác vụ của dự án AmiSoul.

## Cấu trúc Thư mục
- `epic/`: Chứa các Epic (nhóm tính năng lớn) được phân chia theo từng Sprint.
    - [Epics_Backlog.md](file:///d:/Persional_Projects/AmiSoul/docs/project_managements/epic/Epics_Backlog.md): Danh sách toàn bộ các Epic và lộ trình 9 Sprint.
- `ticket/`: Chứa các Task/Ticket chi tiết, được tổ chức theo từng Sprint từ `Sprint-01` đến `Sprint-09`.
    - Mỗi thư mục sprint chứa các file ticket markdown chi tiết cho từng tác vụ kỹ thuật.

## Lộ trình Tổng quát

4. **Giai đoạn 4 (S9):** Tối ưu và Phát hành.
