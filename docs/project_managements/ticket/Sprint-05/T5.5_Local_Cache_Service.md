## 2. Mô tả tác vụ
Triển khai bộ nhớ đệm tại trình duyệt (IndexedDB) để lưu trữ tin nhắn tạm thời, xử lý trạng thái Offline mượt mà và lưu `Device_ID`.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] Sử dụng thư viện `idb` (wrapper cho IndexedDB) để tạo DB.
- [x] Lưu trữ `Device_ID` (tạo tự động bằng UUID) cho luồng Auth-less Onboarding.
- [x] Lưu lịch sử tin nhắn cục bộ. Khởi động ứng dụng (F5): load tin nhắn từ IndexedDB vào `useChatStore` trước khi kết nối Socket để giảm độ trễ hiển thị.
- [x] **Offline State UI:**
  - [x] Không hiện popup lỗi.
  - [x] Chuyển `VibeBackground` sang trạng thái `offline` (Grey → Muted lavender).
  - [x] Đổi Indicator trên Header sang xám kèm text "Đang đợi tín hiệu từ Ami...".
  - [x] Nút Gửi (Input Area) bị mờ (styled as dimmed) nhưng vẫn cho phép bấm để đưa vào Queue.
- [x] Hỗ trợ Queue tin nhắn: Nếu gửi lúc offline, lưu vào Queue, tự động gửi lại khi có mạng.
