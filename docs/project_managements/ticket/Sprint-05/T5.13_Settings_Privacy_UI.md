- [x] **Privacy Controls:**
  - [x] Nút **"Delete Memories"**: Cho phép xóa toàn bộ hoặc chọn lọc ký ức.
  - [x] Toggle **"Incognito Mode"**: Bật/tắt chế độ ẩn danh toàn cầu.
- [x] **Profile Management:** Hiển thị thông tin định danh (Email/ID) và mức độ thân thiết (Bonding Level) hiện tại.
- [x] **Data Export:** Nút tải xuống dữ liệu cá nhân (JSON) theo chuẩn GDPR (chuẩn bị giao diện).
