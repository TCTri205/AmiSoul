## 2. Mô tả tác vụ
Triển khai `VibeBackground` phản chiếu cảm xúc (Session_Vibe) của AI, tạo không gian tĩnh lặng và an toàn qua màu sắc.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Xây dựng Component `VibeBackground` (`position: fixed; z-index: 0`), dùng CSS Mesh Gradient.
- [ ] Ánh xạ 5 trạng thái từ `useVibeStore`:
  - `positive`: Pastel hồng → Lavender → Peach
  - `neutral`: Pastel xanh nhạt → Cream → Lilac
  - `stressed`: Pastel tím trầm → Slate → Dusty blue
  - `crisis`: Soft blue → Pale grey
  - `offline`: Grey → Muted lavender
- [ ] **Quy tắc Transition CHẬM:** Đảm bảo thời gian chuyển màu CSS từ vibe này sang vibe khác tốn **tối thiểu 3-5 giây** (tuyệt đối không chuyển màu ngay lập tức).
- [ ] Tích hợp `Drifting Animation` (hiệu ứng gradient trôi lững lờ chu kỳ 20s) bằng CSS Keyframes.
- [ ] Hỗ trợ Dark Mode: Giảm lightness 30-40%, tăng saturation nhẹ.
