## 2. Mô tả tác vụ
Triển khai `SocketProvider` (React Context) để quản lý một instance duy nhất của Socket.io-client, chịu trách nhiệm truyền tải dữ liệu và tự động phục hồi kết nối.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] Xây dựng `SocketProvider` bọc toàn bộ App.
- [x] Cấu hình Socket.io-client gửi kèm `Device_ID` (từ LocalStorage) hoặc `JWT` trong header.
- [x] Triển khai thuật toán **Auto-reconnect** với Exponential Backoff (max 30s).
- [x] Tích hợp lắng nghe và dispatch events vào Zustand:
  - `stream_chunk` -> `useChatStore.appendChunk()`
  - `vibe_update` -> `useVibeStore.setVibe()`
  - `disconnect` -> `useVibeStore.setConnectionStatus('offline')`
- [x] Cung cấp các methods: `sendMessage()`, `sendInterrupt()` (cho Preemption), `sendReaction()`.
