## 2. Mô tả tác vụ
Xây dựng thành phần UI cốt lõi của chat, xử lý streaming text từ AI, hiệu ứng mượt mà và logic tự động cuộn thông minh.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Xây dựng Component `MessageBubble`: User (căn phải, glassmorphism sáng), AI (căn trái, glassmorphism đậm hơn).
- [ ] Tích hợp **Framer Motion**:
  - `fade-in + slide-up` (User) / `slide-left` (AI).
- [ ] Triển khai Streaming: Render text dần dần theo chunk, kèm con trỏ nhấp nháy ở cuối.
- [ ] **Typing Indicator:** Hiển thị 3 chấm nhảy (bouncing dots) khi mới nhận `processing_start`. Hiển thị "Đã xem ✓" dưới tin user.
- [ ] Tích hợp **React Markdown**: Hỗ trợ render cơ bản.
- [ ] **Smart Auto-scroll:** Tự động cuộn xuống khi có tin mới/stream, NGOẠI TRỪ khi người dùng đang cố ý cuộn lên xem lịch sử (`scrollTop + clientHeight < scrollHeight - 100px`).
- [ ] Hỗ trợ Reply Preview (thanh preview trên Input Area khi trượt để reply).
- [ ] **Timeout Handling:**
  - Nếu > 6s không nhận chunk mới sau `processing_start`: Thay đổi Typing Indicator thành text "Ami đang suy nghĩ...".
  - Nếu > 15s: Hiển thị tin nhắn lỗi nhẹ nhàng "Ami hơi bận, thử lại sau chút nha 💤" kèm nút "Gửi lại".
