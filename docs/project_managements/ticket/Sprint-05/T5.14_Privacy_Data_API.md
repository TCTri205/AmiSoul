## 2. Mô tả tác vụ
Triển khai các API Backend để thực thi quyền "Được lãng quên" (Right to be Forgotten), cho phép xóa vĩnh viễn hoặc tạm thời dữ liệu ký ức và lịch sử hội thoại của người dùng.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] **Hard Delete API:** Xóa vĩnh viễn các bản ghi trong bảng `Memories` và `Sessions` của UserID tương ứng.
- [x] **Soft Delete API:** Đánh dấu `is_deleted` để ẩn khỏi context retrieval nhưng giữ lại log hệ thống (tùy chọn cấu hình).
- [x] **Vector Sync:** Đảm bảo khi xóa bản ghi PostgreSQL, các vector tương ứng trong Redis/Index cũng được dọn dẹp để tránh truy xuất ký ức "ma".
- [x] **Audit Log:** Ghi nhận sự kiện xóa dữ liệu vào hệ thống giám sát an toàn.
