## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Triển khai **MemoryCompressionJob** Worker sử dụng BullMQ.
- [ ] Kích hoạt trigger khi: Session End (30p inactivity), Log Overflow (>500 tin), hoặc Daily Batch (3h sáng).
- [ ] Sử dụng Gemini 1.5 Flash để tóm tắt log chat thành các **Episodic Nodes** (giảm >70% token).
- [ ] Trích xuất Metadata cảm xúc, `Confidence_Score` và các thực thể quan trọng.
- [ ] Tích hợp `EmbeddingService` để tạo vector và lưu vào pgvector.
