## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Cài đặt và cấu hình `@nestjs/bullmq` kết nối thành công tới Redis.
- [ ] Định nghĩa hàng đợi chính: `consolidation_queue`.
- [ ] Tích hợp **Bull Board** (Dashboard) để giám sát trạng thái các job (Completed, Failed, Waiting).
- [ ] Thiết lập cơ chế Retry và Backoff cho các job gặp lỗi tạm thời.
- [ ] Đảm bảo các Worker chạy tách biệt với Main Thread của NestJS.
