## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Tích hợp Gemini Embedding API thành công.
- [ ] Xây dựng `EmbeddingService` để chuyển đổi chuỗi văn bản thành vector.
- [ ] Triển khai cơ chế Cache vector embedding trong Redis.
- [ ] Đảm bảo thời gian tạo embedding < 300ms.
- [ ] **High Availability:** Sử dụng `LlmProvider` (T2.8) để thực hiện embedding, tận dụng khả năng xoay vòng API Key nhằm tránh gián đoạn do Rate Limit.
