## 2. Mô tả tác vụ
Mở rộng khả năng của Prisma Client để hỗ trợ tìm kiếm vector (Vector Search) trên PostgreSQL sử dụng extension `pgvector`, phục vụ cho việc truy xuất ký ức tương tự.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] Kích hoạt thành công extension `vector` trong cơ sở dữ liệu PostgreSQL.
- [x] Cấu hình Prisma schema với bảng `Memories` có cột kiểu `Unsupported("vector(768)")`.
- [x] Triển khai phương thức sử dụng `$queryRaw` để thực hiện tìm kiếm Vector bằng Cosine Similarity.
- [x] Đảm bảo query vector được tối ưu hóa bằng index HNSW (Hierarchical Navigable Small World).
- [x] Kết quả trả về bao gồm nội dung ký ức và điểm số tương ứng (similarity score).
