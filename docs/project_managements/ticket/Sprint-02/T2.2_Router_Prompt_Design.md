## 2. Mô tả tác vụ
Thiết kế System Prompt tối ưu để Gemini trả về kết quả phân tích dưới định dạng JSON cấu trúc, phục vụ cho việc điều phối pipeline.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] Hoàn thiện System Prompt cho Stage 1.
- [x] Đảm bảo Output trả về là JSON hợp lệ với các trường: `intent`, `sentiment`, `complexity`, `urgency`, `identity_anomaly`, `routing_confidence`, `sarcasm_hint`, `timestamp_flag`, `noise_flag`.
- [x] Triển khai **Confidence-Based Routing**: Nếu `routing_confidence` < 0.85, escalate lên Full Cognitive Path.
- [x] Nhận diện được các Intent đặc biệt: `forget_me`, `delete_memory`, `factual_query`.
- [x] Tích hợp logic tóm tắt (Summarization) nếu Message Block > 800 tokens ngay trong prompt này.
