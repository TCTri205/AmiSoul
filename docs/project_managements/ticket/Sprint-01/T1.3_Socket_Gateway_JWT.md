## 2. Mô tả tác vụ
Triển khai Socket.io Gateway để thiết lập kết nối song hướng giữa Client và Server, tích hợp bảo mật qua JWT.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Triển khai `ChatGateway` sử dụng `@nestjs/websockets`.
- [ ] Xây dựng Middleware xác thực JWT cho các kết nối Socket.
- [ ] Hỗ trợ nhận diện và xử lý metadata `session_type` (Incognito vs Persistent) từ client.
- [ ] Xử lý các sự kiện cơ bản: `connection`, `disconnect`, và `message_sent`.
- [ ] Đảm bảo độ trễ gateway phản hồi sự kiện < 50ms.
