## 2. Mô tả tác vụ
Thiết kế và triển khai cấu trúc cơ sở dữ liệu sử dụng Prisma ORM. Thiết lập hệ thống DB cơ bản qua Docker Compose.

## 3. Tiêu chí chấp nhận (Chi tiết)
- [ ] Thiết lập `docker-compose.yml` chạy PostgreSQL (với extension `pgvector`) và Redis.
- [ ] Cấu hình Prisma Client và thực hiện kết nối thành công đến database.
- [ ] Định nghĩa Schema Prisma cho các bảng core:
    - `User`: Thông tin người dùng cơ bản.
    - `Account`: Thông tin định danh.
    - `Session`: Chứa flag `is_incognito` (kiểu Boolean).
    - `Message`: Lưu trữ nội dung tin nhắn và metadata.
- [ ] Chạy thành công lệnh `prisma migrate dev` để khởi tạo database.
