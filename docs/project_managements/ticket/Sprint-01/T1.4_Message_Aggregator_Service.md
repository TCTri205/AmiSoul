## 2. Mô tả tác vụ
Xây dựng dịch vụ điều phối việc gom tin nhắn (Aggregator) và xử lý logic theo chế độ ẩn danh (Incognito).

## 3. Tiêu chí chấp nhận (Chi tiết)
- [x] Triển khai `MessageAggregatorService` trong NestJS.
- [x] Xây dựng logic phân luồng xử lý dựa trên flag `is_incognito`:
    - Nếu `true`: Chỉ lưu trữ tạm thời trong Redis, không lưu vào database bền vững.
    - Nếu `false`: Chuẩn bị dữ liệu để lưu vào PostgreSQL.
- [x] Tích hợp với Redis để quản lý buffer tin nhắn của từng người dùng.
- [x] Đảm bảo tính nguyên tử (Atomicity) khi cập nhật buffer trong Redis.
